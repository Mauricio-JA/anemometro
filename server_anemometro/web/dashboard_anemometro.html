Resultado Esperando datos
Si BGT < 25°C  HLI = 10.66 + 0.28·HR + 1.3·BGT − WS
Si BGT ≥ 25°C  HLI = 8.62 + 0.38·HR + 1.55·BGT − 0.5·WS + e(2.4−WS) WS = viento real (tu anemómetro) · HR = humedad relativa (%), ingreso manual · BGT = temp. de globo negro (°C), ingreso manual — el sensor instalado es un BMP180 (confirmado por su chip ID), que mide temperatura y presión, pero no humedad ni globo negro. Ambos valores los escribe quien opera el panel. Fórmula: Gaughan et al., 2008. Umbral oficial de ganancia de calor para Bos indicus 100% (Nelore): HLI 96. La franja 75–96 es un margen de alerta temprana propio del proyecto, antes de alcanzar el umbral validado por el paper.
